## How DOIs are minted

**Per-paper DOIs:** each PDF is uploaded as its own Zenodo publication (report). Share those links in posts.

**Series archive DOI:** enable this repo under [Zenodo → GitHub](https://zenodo.org/account/settings/github/), then publish a GitHub **Release**. Zenodo archives the release ZIP and assigns one DOI for that snapshot.
