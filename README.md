## How DOIs are minted

1. Enable this repo under [Zenodo → GitHub](https://zenodo.org/account/settings/github/).
2. Publish a GitHub **Release** (tag `vYYYY.MM.DD` or `v1.0.0`).
3. Zenodo archives the release ZIP and assigns a DOI.
